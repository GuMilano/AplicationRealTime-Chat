# AplicationRealTime-Chat

Aplicação de **chat em tempo real** com suporte a múltiplas salas, troca de mensagens instantânea e atualização ao vivo para todos os usuários conectados.

> Projeto voltado para estudo e prática de conceitos de **comunicação em tempo real**, eventos, broadcast e organização de um app de chat.

---

## ✨ Funcionalidades

- ✅ Envio e recebimento de mensagens em tempo real
- ✅ Salas (rooms) / canais de conversa
- ✅ Lista de usuários conectados (opcional)
- ✅ Notificação de entrada/saída de usuários (opcional)
- ✅ Interface web simples para chat
- ✅ Backend com servidor em tempo real (WebSocket/Socket)
